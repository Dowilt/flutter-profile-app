# Profile App — Лабораторная работа №2

Flutter веб-приложение «Профиль пользователя» с тремя вкладками, переключением темы и интерактивными элементами.

## Описание

Приложение реализует все самостоятельные задания из лабораторной работы №2:

### Задание 1 (базовое) — Персонализация профиля
- Данные профиля заменены на персональные (имя, контакты, интересы)
- Цветовая тема изменена на `deepPurple`
- Добавлена дополнительная карточка с информацией (Университет, GitHub)

### Задание 2 (среднее) — Переключение темы
- `MyApp` переведён на `StatefulWidget`
- Реализовано переключение светлой/тёмной темы через `IconButton` в AppBar
- Функция переключения передаётся из `MyApp` в `ProfileScreen` через параметры конструктора

### Задание 3 (продвинутое) — Три вкладки
- **Профиль** — основной экран с аватаром, карточками контактов, тегами интересов и кнопками
- **Галерея** — сетка изображений через `GridView.count` с `Image.network`
- **Контакты** — список контактов через `ListView.builder` (5 контактов)
- Навигация между вкладками через `BottomNavigationBar`

## Технологии

- Flutter (Web)
- Dart
- Nginx (для раздачи статики)
- Docker

## Запуск

```bash
docker-compose up -d --build
```

Приложение будет доступно по адресу: http://localhost:8080

## Остановка

```bash
docker-compose down
```

## Демонстрация

Для записи демонстрации:
1. Запустите приложение командой `docker-compose up -d --build`
2. Откройте http://localhost:8080 в браузере
3. Запишите экран, показывая:
   - Переключение между вкладками (Профиль, Галерея, Контакты)
   - Нажатие кнопки «Нравится» (счётчик увеличивается)
   - Нажатие кнопки «Написать» (SnackBar-уведомление)
   - Переключение темы (светлая/тёмная)
   - Переход на экран «О приложении» через иконку (i)
